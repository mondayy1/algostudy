### #백준 2438번: 별찍기 1

중첩 반복문을 사용하여 첫째 줄부터 입력받은 N번째 줄까지 차례대로 별을 출력한다.
코드에서 유의할 점은 다음과 같다.
- 하나씩 늘리며 출력하므로 증감식에서 ++를 사용
- 개행문자는 안쪽 for문이 끝나고 난 뒤 바깥 for문에서 출력

### #백준 2438번: 별찍기 1

중첩 반복문을 사용하여 첫째 줄부터 입력받은 N번째 줄까지 차례대로 별을 출력한다.
단, 아래와 같이 출력할 것
    *
   **
  ***
 ****
*****
코드에서 유의할 점은 다음과 같다.
- 중첩 반복문의 안쪽 for문이 두 개가 사용됨.
- 안쪽 for문은 바깥 for문의 i를 기준으로 돌아감.
- 띄어쓰기와 * 은 반비례함.
