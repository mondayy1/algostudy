## Commit Message
|*Type*|*Subject*|
|:---|:---|
|**[ac]**|해결|
|**[wa]**|못품|
|**[tle]**|시간초과|
|**[docs]**|문서 (문서 추가, 수정, 삭제)|
|**[chore]**|빌드, 설정 파일|

ex) [Type] #이슈번호 커밋메세지 -> "git commit -m "[ac]#12 백준 1237   
[ac] 태그를 달 수 있을 때까지 커밋하기   
**만약 결과가 [wa]이거나 [tle]라면 그 원인을 찾아서 해결하기**

</br>
