## Commit Message
|*Type*|*Subject*|
|:---|:---|
|**[ac]**|해결|
|**[wa]**|못품|
|**[tle]**|시간초과|
|**[docs]**|문서 (문서 추가, 수정, 삭제)|
|**[chore]**|빌드, 설정 파일|

ex) [Type] #이슈번호 커밋메세지 -> "git commit -m "[ac]#12 백준 1237"
